The Philosophy
Your ancestors built empires. You cannot even close 3 Chrome tabs.
— Orbiton, during motivate me

We believe your computer should obey you, not the other way around. Orbiton was built on a simple conviction: the gap between thought and action on a computer is too wide. You should not have to break flow state to click through menus, hunt for folders, or type the same search query for the hundredth time.

Every command is processed locally. Your voice never leaves your machine. There are no API keys to manage, no cloud dashboards to monitor, and no privacy policy written by lawyers you will never meet. Just a Python process listening through your headset, waiting for the word TOKYO.

We are Kosmosic. We build tools that make your workspace smarter, faster, and slightly more judgmental.

Capabilities

Everything within earshot.

Orbiton is not a chatbot. It is a command layer between your voice and your operating system.

🎤

Wake Word Activation

Say "TOKYO" and Orbiton snaps to attention. No buttons. No clicks. Just your voice cutting through the noise. Configurable wake word via CONFIG["wake_word"].

🧠

Neuro-Link Intelligence

Built-in NLP handles homophones, spoken math, and context routing. "exambored" becomes "exam mode". "two times two" becomes 2*2. It understands you even when you mumble.

🔒

Zero Cloud, Zero Telemetry

Everything runs locally. Your voice never leaves your machine. No API keys to manage. No privacy policy to dread. The only network requests are the ones you explicitly command — like opening a Google search.

⚡

System Native

Open VS Code projects, launch Chrome searches, navigate files, check weather, track flights — all through OS-level integration. Windows Explorer, macOS open, Linux xdg-open. It speaks your OS's language.

🗣️

Edge TTS Voices

Orbiton talks back using neural text-to-speech. Aria, Jenny, or any Edge voice you prefer. It sounds human because it is. Configure your voice profile in CONFIG["voice"].

🐍

Python-Powered & Hackable

Written in pure Python. Add commands in minutes. The entire architecture is a single file away from your next great idea. No compile steps. No build chains. Just edit and run.

🧮

Spoken Math Engine

The MathNormalizer converts spoken expressions into safe Python before evaluation. "twenty five divided by five" → 25/5. "three squared" → 3**2. No more mental arithmetic.

📚

Live Knowledge Lookup

Wikimedia scraping with local caching. Ask "tell me about constellations" or "what is the capital of Mongolia" — Orbiton queries Wikipedia and Wiktionary live, caches results, and reads them back to you.

💪

Toxic Motivation Roast

Sometimes you need a push, not a hug. Say "motivate me" and Orbiton deploys industrial-grade accountability. "Your GitHub contribution graph looks like a deforestation map. Embarrassing."

Architecture

Built to be understood.

Orbiton's architecture is intentionally flat. Two core files. One purpose. Zero indirection.

Intelligence Module

It understands you, even when you do not.

Orbiton's pluggable intelligence system handles the messy reality of spoken language.

📝 NLP Normalization

Contractions, slang, and lazy phrasing are resolved before routing. "what's" → "what is". "exam board" → "exam mode". You do not need to enunciate like a robot.

🔊 Homophone Correction

Speech recognition is imperfect. Orbiton maps common misheard phrases to intended commands. "exambored" routes to "exam mode". "you too" does not accidentally open YouTube.

📐 Math Normalization

The MathNormalizer converts spoken math into safe Python expressions before evaluation. "two times" → *. "divided by" → /. "x" → *. No eval() on raw strings.

🌐 Knowledge Lookup

Live Wikipedia and Wiktionary queries with local JSON caching. Ask about constellations, moon phases, aviation facts, or space trivia — answers are scraped, cached, and spoken back.

🧭 Contextual Routing

Orbiton decides whether to search Google, calculate locally, open a file, or answer directly based on command structure and intent — not just keyword matching.

💾 Persistent Memory

User information stored in ~/.neuro_link_memory.json. Orbiton remembers who you are, your preferences, and your session history across reboots.

The Future

Model classes.

Seven generations, one evolution. Each class represents a leap in intelligence, autonomy, and integration.

Live Now — v0.x → v1.x

Generation 1 — Tokyo

Basic reasoning, voice commands, web search, file management, aviation tools, and the toxic motivation engine. The foundation. Fast, local, and brutally honest.

Planned — v2.x

Generation 2 — Odyssey

Large-scale growth. Advanced reasoning, local LLM integration, long-term memory, and a personalization engine. The beginning of true intelligence.

Planned — v3.x

Generation 3 — Genesis

Agentic behavior and true automation. Long-running tasks, predictive execution, multi-step workflows, and a scraper living on your PC with OTA intel updates.

Planned — v4.x (ROI-dependent)

Generation 4 — Micron

Lite version for constrained environments. All core functionality stripped of heavy dependencies, for older hardware or minimal installs. Only built if Odyssey and Genesis prove ROI.

Planned — v5.x

Generation 5 — Aphrodite

Expansion beyond the original vision. IoT integration, multi-language support, various voices, wake word customization, PC management, email, calls, and calendar.

Vision — v6.x

Generation 6 — Singularity

Full autonomy. The second user of your computer. Multi-device ecosystem, predictive everything, self-maintenance, multi-device continuity, wearables, smart glasses, and SBCs.

Final Vision — v7.x

Generation 7 — Utopia

Final evolution. The absolute top. Orbiton becomes an ecosystem. Replaces bloatware AIs. Lives on every device you own. Knows what you need before you ask.
